## 概要

遊べるミュージックビデオゲーム第4作目のプロトタイプをつくりました。  
「リズムゲーム」「アクションゲーム」「STG」と続いてきたこのシリーズですが、  
今回のジャンルは「落ちものパズルゲーム」となっています。

ゲームはこちらから遊べます（PC・スマホ両対応）  
https://kakothk.github.io/vocagame4-prototype/


## 操作方法

Z：左回転  
X：右回転  
矢印左右キー：横移動  
矢印上キー　：ハードドロップ  

※スマホの場合は画面下のパネルで操作してください。


## あそびかた

まず、画面左上の再生マークのボタンを押してみてください。  
ゲームがスタートします。

空からいろいろな色のブロックがリズムにノって降ってきます。  
とりあえず、同じ色のブロックを縦か横、どちらでもいいので隣り合わせになるように落としてみてください。  

同じ色のブロックを隣り合わせにすると、ブロックに「♩」マークがつきます。  
画面下の「NEXT BIND」バーが満タンになるまでに、とにかくこの「♩」をたくさん作ってください。

「NEXT BIND」バーはBGMの16小節のサイクルで満タンになります。  
このとき、今まで作ってきた「♩」ブロックがすべて消えていきます。  
「♩」ブロックをたくさん作って一気に消す……これを繰り返していくゲームです。


## ノルマについて

画面左下に「LEVEL」「NORMA」「BIND」という数値が見えると思います。  
「LEVEL」は現在の難易度、「NORMA」は必要ノルマ、「BIND」は完成している「♩」ブロックの数を表します。 

「♩」ブロックが消えるとき、「♩」ブロック1個につき、ノルマを1減らしてくれます。  
ノルマを達成できるように、時間内にとにかくたくさんの「♩」ブロックを作ってください。  
ノルマはLEVELが上がるにつれて、どんどん増えていきます。  


## オジャマな「ノイズブロック」

ノルマを達成できなかったとき、未達成ノルマ分の「ノイズブロック」が空から降ってきます。  
この「ノイズブロック」は、同じ色で重ねても「♩」になりません。  
無駄に盤面のスペースを埋めていく、オジャマなブロックです。  

ただ、「♩」ブロックが隣にあるとき、誘爆するようにこの「ノイズブロック」を消してくれます。  
しかも、ただの「♩」ブロックと違って「ノイズブロック」を消すとノルマを2も減らしてくれます。  
うまく「ノイズブロック」を消せると、ピンチをチャンスに変えられるかも……  


## 強力な「白ブロック」

同じ色のブロックを縦と横2マスずつ（2x2）並べたとき、  
もしくは同じ色のブロックを縦4マス、横4マスずつ（1x4/4x1）並べたとき、  
その4つのブロックが全て「白ブロック」に変化します。

この「白ブロック」はとても強力です。  
「白ブロック」は、上下隣り合うブロックをすべて強制的に「♩」ブロックに変えることが出来ます。  
しかも「ノイズブロック」すら「♩」ブロックに変えることが出来ます。  
うまく「白ブロック」を使いこなせれば、たくさんの「♩」ブロックを作ることが出来ます。  

それと「白ブロック」は、わざわざ自分で作り出さなくても、たまに「NEXT」で出てくることもあります。  
さらにノルマを余分に達成できたとき、「ノイズブロック」の代わりに「白ブロック」が降ってきます。  
つまりノルマを必要以上に達成できれば次のレベルが楽になるということですね。

## その他、（雑に）こまかい仕様と説明と解説

■ こまかい仕様  

・ノルマ未達成でも、1つだけオマケで「♩」ブロックが降ってきます。  
　それを基点にして、逆転する事もできるかも。  

・ノルマを余分に達成した時のボーナス「♩」ブロックは、ノルマ-5ごとに1つ追加されます。  
　（ 0 ~ -4 : 「♩」1個が降ってくる、 -5 ~ -9：「♩」が2個降ってくる、 -10 ~ -14：「♩」が3個降ってくる … ）  

・NEXTの「♩」ブロックは、10回に1回必ず出現するという最低保証がついています。  

■ オプションボタン  

・画面下部の操作パネルにある「MODE」「CHORD」「BEAT」は、オプションボタンになります。  

・「MODE」は、ノルマ未達成で強制ゲームオーバーになる「ノーマルモード」と、  
　ノルマを達成しても続いていく「エンドレスモード」があります（デフォルトはエンドレス）。  

・「CHORD」と「BEAT」は、後ろで鳴っているコードとビートをミュートすることが出来ます。  
　好きな音楽を流しながら遊んでください。  

・画面一番上にある「BPMバー」を操作すると、ゲームスピードを調整できます。  
　個人的には150がいいかなと思うのですがどうでしょう。  

■ 解説  

・このゲームは「ルミナス」「ぷよぷよ」「テトリスエフェクト」  
　「マジカルビーツ」「ウィッチパズリズム」あたりを参考にしています。  

・「♩」ブロックを16小節ごとにまとめて消すというルールは、  
　ゲームオーバーへ近づくリスクを貯めて貯めて一気に開放することで、  
　爽快感を得られるんじゃないかという発想で考えました。  

・それと、このゲームは最終的には初音ミクが歌う「歌」になるので、  
　セクションの区切りでゲームが進行する事を意識しています。  
　作る曲の構成によっては、16小節固定になるとは限らないかも。  

・ゲームは、MVとしてのストーリー性を意識したゲームクリアが存在する「ノーマルモード」と、  
　ストイックにスコアを競う「エンドレスモード」、2つのモードを用意する予定です。

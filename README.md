## ・概要

遊べるミュージックビデオゲーム第4作目のプロトタイプをつくりました。  
「リズムゲーム」「アクションゲーム」「STG」と続いてきたこのシリーズですが、  
今回のジャンルは「落ちものパズルゲーム」となっています。

ゲームはこちらから遊べます（PC・スマホ両対応）  
https://kakothk.github.io/vocagame4-prototype/


## ・操作方法

Z：左回転  
X：右回転  
矢印左右キー：横移動  
矢印上キー　：ハードドロップ  

※スマホの場合は画面下のパネルで操作してください。


## ・あそびかた

まず、画面左上の再生マークのボタンを押してみてください。  
ゲームがスタートします。

空からいろいろな色のブロックがリズムにノって降ってきます。  
とりあえず、同じ色のブロックをタテかヨコ、どちらでもいいので隣にくっつけるように落としてみてください。  

隣に同じ色のブロックをくっつけると、ブロックに「♩」マークがつきます。  
画面下の「NEXT BIND」バーが満タンになるまでに、とにかくこの「♩」をたくさん作ってください。

「NEXT BIND」バーは16小節のタイミングで満タンになります。  
このとき、今まで作ってきた「♩」ブロックがすべて消えていきます。  
「♩」ブロックをたくさん作って一気に消す……これを繰り返していくゲームです。


## ・ノルマについて

画面左下に「LEVEL」「NORMA」「BIND」という数値が見えると思います。  
「LEVEL」は現在の難易度、「NORMA」は必要ノルマ、「BIND」は完成している「♩」ブロックの数を表します。 

「♩」ブロックが消えるとき、「♩」ブロック1個につき、ノルマを1減らしてくれます。  
ノルマはLEVELが上がるにつれて、どんどん増えていきます。  
ノルマを達成できるように、時間内にとにかくたくさんの「♩」ブロックを作ってください。


## ・オジャマな「ノイズブロック」

ノルマを達成できなかったとき、未達成な残りノルマ分の「ノイズブロック」が空から降ってきます。  
この「ノイズブロック」は、同じ色で重なっても「♩」になりません。  

ただ、「♩」ブロックがとなりにあるとき、誘爆するようにこの「ノイズブロック」を消してくれます。  
しかも、ふつうの「♩」ブロックと違って、「ノイズブロック」を消すとノルマを2も減らしてくれます。  
うまく「ノイズブロック」を消せると、ピンチをチャンスに変えられるかもしれません。


## ・強力な「白ブロック」

同じ色のブロックを縦と横2マスずつ（2x2）並べたとき、  
もしくは同じ色のブロックを縦4マス、横4マスずつ（1x4/4x1）並べたとき、  
その4つのブロックが全て「白ブロック」に変化します。

この「白ブロック」はとても強力です。  
「白ブロック」は、上下隣り合うブロックをすべて強制的に「♩」ブロックに変えることが出来ます。
しかも「ノイズブロック」すら「♩」ブロックにすることが出来ます。  
うまく「白ブロック」を使いこなせれば、たくさんの「♩」ブロックを作ることが出来ます。  

……あと「白ブロック」は、わざわざ自分で作り出さなくても、たまに「NEXT」で出てくることもあります。  
それと、ノルマを余分に達成できたとき、「ノイズブロック」の代わりに「白ブロック」が降ってきます。  
なので、ノルマを過剰に達成できれば次のレベルが楽になるということですね。  


## ・ゲームオーバー

画面上部の「××」までブロックが埋まってしまうと、ゲームオーバーとなります。  
どこまで生き残れるか試してみてくださいね。  

ちなみに、これが自分の最高スコアです（BPM:150開始）
